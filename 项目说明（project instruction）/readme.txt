1.此项目基于Maven web工程搭建,请确保有Maven环境.
2.运行项目说明下的secondhandtradingnetwork.sql建立数据库及其依赖.
3.开发工具导入工程.
4.修改main/resources目录下的jdbc.properties文件，改为自己的mysql用户名和密码.
5.启动工程，浏览器下输入http://localhost:8080/secondHandTradingNetwork/secondHandTradingNetwork/login
